**O que é Flutter?**

Uma ferramenta que permite construir **aplicações nativas cross-platform(iOS, Android)** usando um **único código fonte/linguagem de programação.**

**A linguagem de programação por trás do Flutter é o Dart.**

**O flutter é formado por:**

- **SDK (Software Development Kit)** - Ferramentas para compilar seu código para código nativo + desenvolver com facilidade.

- **Framework/Widget Library** - Componente de UI reutilizáveis (= widgets), funções utilitárias, pacotes.

---

**O que é Dart?**

Linguagem de desenvolvimento criado pelo **Google**.

Focada em frontend (aplicações mobile, web) desenvolvimento de interface de usuário (UI)

Linguagem orientada a objetos e Tipagem forte.

Sintaxe é mistura de Java, Javascript, C#.

**O Flutter é um mixer das ferramentas de desenvolvimento JBK e também do próprio Framework (Flutter Framework).**

**O framework é desenvolvido em Dart para programar, criar lógica, gerenciar os estágios de aplicação, para usar os Widgets.**

**O Flutter é um conjunto de recursos API para que possa desenvolver as aplicações Cross-platform**

**À partir do Dart, criamos os componentes, widgets, componentes visuais, funções utilitárias.**

**Tanto o próprio Flutter utiliza Dart como linguagem de programação como também utilizaremos Dart para criar aplicações usando o Flutter**

A outra parte do Flutter é o **Flutter SDK**, um conjunto de ferramentas necessárias para desenvolver as aplicações no geral.
