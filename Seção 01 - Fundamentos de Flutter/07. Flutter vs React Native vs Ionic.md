**Flutter e suas alternativas:**

**Flutter:**

- *O flutter tem um ideal para arquitetura de criação, sendo atualmente a mais adotada.**

- Dart + Flutter: Linguagem e integração com Widgets, com comunidade impulsionando novos recursos.

- App Nativa compilada: O código Dart é compilado para linguagens específicas do iOS e Android.

- NÃO utiliza os componentes nativos do iOS/Android.

- Cross-Platform (Mobile, Web, Desktop): As 3 vertentes serão integradas ao Framework.

- Desenvolvido pela Google.
 
**React Native:**

- Javascript / React.js: São linguagens conhecidas e muito utilizadas.

- App Nativa Parcialmente Compilada (Componentes Visuais): Utiliza os componentes visuais Nativos de cada plataforma.

Exemplo: 

**Usa os componentes visuais nativos, mas o código JavaScript será interpretado para na hora que a aplicação estiver rodando, fazer as chamadas nativas.**

- Utiliza os componentes nativos do iOS/Android.

- Foco em Mobile.

- Desenvolvido pelo Facebook.

**Ionic:**

- Javascript (Qualquer ou nenhum framework.

- Utiliza HTML, CSS e JavaScript.

- App dentro de uma WebView: Não é uma aplicação nativa, é uma aplicação híbrida.

- NÃO utiliza os componentes nativos do iOS / Android.

- Cross-Platform (Mobile, Web e Desktop).

- Desenvolvido pela Ionic
