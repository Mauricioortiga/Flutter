O código Dart é um fundamento de base para desenvolver e utilizar nossos frameworks baseados em Dart.

A API do Flutter contém utilitários e Widgets utilizando Dart.

O nosso próprio código em Dart acessa a API do Flutter

O processo de compilação realiza um processo de conversão de Dart para código nativo (iOS ou Android).

**Esse processo é feito pelo Flutter SDK (Converter de Dart para código nativo).**
